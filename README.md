# Assistente-de-Voz-Multi-Idiomas-Com-Whisper-e-ChatGPT-Python-JavaScript-.
Desenvolvi um código de programação para um "assistente virtual" utilizando IA, JavaScript e Python; o projeto consiste em você registrar sua voz e através de códigos e API's, obter sua pergunta em "Speak to text" e uma resposta pela IA escrita e convertida para "Text to speak".
